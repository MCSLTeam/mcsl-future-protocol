# 数据模型

数据模型是在 [](action.md) 和 [](event.md) 中常用的类型。

## MessagePack 拓展

### uuid {#uuid}

- 数据编号：`0`（`0x00`）
- 数据类型：`fixext16`
- 数据说明：用于存储 UUID v4 数据
- 数据格式：

```
+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+
|  0xd8  |  0x00  | UUID 数据，二进制大端序存储，16 字节长度                                                                                                           |
+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+--------+
```
